# DROP BOX CHECK!

Dropbox のプロジェクトフォルダ（進行表・音源・歌詞・図面など）を、一画面でズラッと表示してチェックするためのコンサート/ライブ向けビューア。

**URL:** https://nrs2013.github.io/nrs-check/

## できること

- ローカル同期 Dropbox フォルダをブラウザで選ぶと、中身がカラム並列で表示
- 音源（WAV/MP3/AIFF/FLAC等）はクリックで即再生 + キーボードで曲送り
- 画像・動画・PDF はサムネ + プレビュー
- サムネ表示 / LIST 表示 切替

## キーボードショートカット

| キー | 動作 |
|---|---|
| `Space` | 再生 / 停止 |
| `↑` / `↓` | ファイル切替 |
| `←` / `→` | 音源シーク（5秒） |

## 対応ブラウザ

ローカルフォルダ選択に File System Access API を使うため、**Chrome / Edge / Arc などの Chromium 系ブラウザ**でのみ動作します。

## デザイン

Mail-Check! の warm dark + 琥珀（#d4925a）アクセントを継承。
